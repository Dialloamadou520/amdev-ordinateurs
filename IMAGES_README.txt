============================================
AJOUTER DES IMAGES AUX PRODUITS
============================================

NOUVEAU CATALOGUE - 11 ORDINATEURS (Sans Apple)

1. DOSSIER IMAGES
-----------------
Créez un dossier "images" à côté de vos fichiers HTML.

Structure:
EtuOrdinateur/
├── index.html (page principale)
├── admin.html
└── images/
    ├── hp-pavilion-15.jpg
    ├── hp-elitebook-840.jpg
    ├── hp-probook-450.jpg
    ├── lenovo-thinkpad-e15.jpg
    ├── lenovo-thinkpad-t14.jpg
    ├── lenovo-ideapad-3.jpg
    ├── dell-latitude-5520.jpg
    ├── dell-inspiron-15.jpg
    ├── dell-optiplex-desktop.jpg
    ├── asus-rog-strix.jpg
    └── acer-aspire-5.jpg


2. FORMATS SUPPORTÉS
--------------------
- JPG / JPEG
- PNG
- WebP (recommandé pour la performance)

Taille recommandée: 800x500 pixels (16:10)


3. AJOUTER UN PRODUIT AVEC IMAGE
--------------------------------
1. Allez sur admin.html
2. Connectez-vous (mot de passe: amdev2026)
3. Cliquez "Ajouter"
4. Remplissez le formulaire:
   - Nom: HP Pavilion 15
   - URL de l'image: images/hp-pavilion-15.jpg
   - Spécifications: Intel Core i5 · 8 Go RAM · 512 Go SSD
   - Caractéristiques: Ajoutez les tags (Écran 15.6", Windows 11...)
5. Cliquez "Enregistrer"


4. OBTENIR DES IMAGES
-------------------
Option A - Photos réelles:
- Prenez des photos de vos ordinateurs
- Redimensionnez à ~800x500px
- Enregistrez dans le dossier images/

Option B - Images en ligne:
- Téléchargez depuis le site du fabricant (HP, Dell, etc.)
- Ou utilisez des sites comme unsplash.com (gratuit)

Option C - Placeholders temporaires:
- https://via.placeholder.com/800x500?text=HP+Pavilion
- https://placehold.co/800x500?text=MacBook


5. IMPORTANT
------------
- Les images doivent être dans le dossier "images/"
- Utilisez des noms simples sans espaces (remplacez par -)
- Le chemin dans l'admin doit être: images/nom-du-produit.jpg


6. EXEMPLE DE CHEMIN
--------------------
Si votre image est: EtuOrdinateur/images/mon-pc.jpg
Dans l'admin, mettez: images/mon-pc.jpg
